# Project 2: Basic Encryption & Decryption

**Industrial Training | DecodeLabs | Batch 2026**

## About
Caesar Cipher implementation in Python that encrypts and decrypts text using a shift key.

## How to Run
```bash
python3 caesar_cipher.py
```

## Example
```
Input     : Hello World
Shift Key : 3
Encrypted : Khoor Zruog
Decrypted : Hello World
```
